Le workflow fondamental sur **Upwork**, de la création de l'offre jusqu'au paiement du client, suit généralement ce parcours :

```text
1. Recherche d'un projet
        │
        ▼
2. Analyse de l'annonce
   - Comprendre le besoin
   - Vérifier le budget
   - Évaluer la faisabilité
        │
        ▼
3. Création de la proposition (Offer/Proposal)
   - Message personnalisé
   - Prix proposé
   - Délais
   - Portfolio et réponses aux questions
   - Envoi avec des Connects
        │
        ▼
4. Attente de la réponse du client
        │
   ┌────┴────┐
   │         │
Refus     Intérêt
             │
             ▼
5. Discussion (Messages)
   - Questions/Réponses
   - Clarification du projet
   - Négociation du prix et du délai
        │
        ▼
6. Entretien (optionnel)
   - Chat
   - Appel audio
   - Appel vidéo
        │
        ▼
7. Offre officielle du client
   (Contract Offer -> lecture du draft du contrat)
        │
        ▼
8. Acceptation du contrat via la double signature
        │
        ▼
9. Financement
   ┌───────────────────────┐
   │           │
   │ → Le client finance   │
   │   l'escrow            │
   └───────────────────────┘
                      │
        ▼
   ┌───────────────────────┐
   │                 │
   │ → Contrat actif  │
   │      │
   └───────────────────────┘
        │
        ▼
     ┌───────────────────────┐
   │                 │
   │ → Ouverture espace de travail │
   │      │
   └───────────────────────┘      
10. Réalisation du travail
   - Développement
   - Design
   - Rédaction
   - etc.
        │
        ▼
11. Livraison
   - Soumission des fichiers
   - Demande de paiement (Fixed Price)
        │
        ▼
12. Révision (si nécessaire)
   - Corrections
   - Nouvelle livraison
        │
        ▼
13. Approbation du client
        │
        ▼
14. Paiement
   Fixed Price :
      Escrow → Débloqué
   Hourly :
      Paiement automatique après la période de revue
        │
        ▼
15. Fonds disponibles
   (avec le délai de sécurité d'Upwork)
        │
        ▼
16. Retrait des fonds
   - Compte bancaire
   - Payoneer
   - Autres méthodes disponibles selon le pays
        │
        ▼
17. Clôture du contrat
        │
        ▼
18. Évaluations mutuelles
   - Client → Freelancer
   - Freelancer → Client
```

### Résumé simplifié

1. Trouver une mission.
2. Envoyer une proposition.
3. Échanger avec le client.
4. Recevoir et accepter le contrat.
5. Le client sécurise le paiement (escrow pour un forfait) ou active un contrat horaire.
6. Réaliser le travail.
7. Livrer.
8. Le client valide.
9. Upwork libère le paiement.
10. Retirer les fonds.
11. Les deux parties laissent une évaluation.

Ce flux couvre le cycle de vie standard d'une mission sur Upwork, avec des variantes selon qu'il s'agit d'un contrat **à prix fixe (Fixed Price)** ou **horaire (Hourly)**.
